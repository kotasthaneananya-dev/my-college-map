# my-college-map
This project is an interactive map designed to display landmarks (Educational, Religious, Historical, Natural). It is built to be generalized, meaning it can be adapted for any location by simply updating the data file. # my-college-map

An interactive campus navigation system built with React, TypeScript, and Vite. This project focuses on Data Generalization, organizing complex campus landmarks into easy-to-navigate categories.

Key Features

* Dynamic Markers: Interactive points of interest across the campus.
* Data Generalization: Landmarks are categorized to reduce visual clutter:
* Educational: Departments, Labs, and Libraries.
* Religious: Temples and meditation zones.
* Historical: Heritage sites and statues.
* Natural: Gardens, parks, and open green spaces.


* Responsive Design: Optimized for both mobile and desktop views.

Tech Stack

* Frontend: React.js with TypeScript
* Styling: Tailwind CSS
* Build Tool: Vite
* Data Structure: JSON-based landmark management

 Project Structure

* src/: Core React components and map logic.
* client/: Frontend assets and configuration.
* server/: Backend/API logic.
* shared/: Common types and utility functions.
